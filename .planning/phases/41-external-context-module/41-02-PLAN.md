    **ContextModule.csproj** — Mirror PortModule.csproj structure:
    - net8.0, ImplicitUsings, Nullable
    - ProjectReference to `../../src/OpenAnima.Contracts/OpenAnima.Contracts.csproj` with `<Private>false</Private>`
    - MSBuild `PackageOamod` target AfterTargets="Build":
      1. Create a staging directory `$(OutputPath)_oamod_staging/`
      2. Copy only `ContextModule.dll` and `module.json` to staging
      3. ZipDirectory staging to `$(OutputPath)../ContextModule.oamod`
      4. Copy .oamod to `$(SolutionDir)src/OpenAnima.Core/modules/`
      5. Clean up staging directory
    - Note: Use `$(MSBuildThisFileDirectory)` for reliable path resolution to module.json

    **ContextModule.cs** — Full IModule implementation:
    ```
    namespace ContextModule;

    [InputPort("userMessage", PortType.Text)]
    [InputPort("llmResponse", PortType.Text)]
    [OutputPort("messages", PortType.Text)]
    [OutputPort("displayHistory", PortType.Text)]
    public class ContextModule : IModule
    ```

    Constructor: optional params for IModuleConfig, IModuleContext, IModuleStorage, IEventBus, ILogger (all nullable with default null).

    Fields: `_subscriptions` (List of IDisposable), `_history` (List of ChatMessageInput), `_historyPath` (string?).

    **InitializeAsync:**
    1. Resolve history path: if `_storage` not null, `_historyPath = Path.Combine(_storage.GetDataDirectory(), "history.json")`
    2. Load existing history: if `_historyPath` exists and file exists, read and deserialize via `ChatMessageInput.DeserializeList`
    3. Subscribe to `"ContextModule.port.userMessage"` → `HandleUserMessageAsync`
    4. Subscribe to `"ContextModule.port.llmResponse"` → `HandleLlmResponseAsync`

    **HandleUserMessageAsync(string payload, CancellationToken ct):**
    1. Append `new ChatMessageInput("user", payload)` to `_history`
    2. Build output list: if system message configured (non-empty), prepend `new ChatMessageInput("system", systemMessage)` to a copy of `_history`
    3. Publish to `"ContextModule.port.messages"` with `ChatMessageInput.SerializeList(outputList)`, SourceModuleId = "ContextModule"

    **HandleLlmResponseAsync(string payload, CancellationToken ct):**
    1. Append `new ChatMessageInput("assistant", payload)` to `_history`
    2. Persist: if `_historyPath` not null, `await File.WriteAllTextAsync(_historyPath, ChatMessageInput.SerializeList(_history))`
    3. Build output list (same system message prepend logic as above)
    4. Publish to `"ContextModule.port.displayHistory"` with `ChatMessageInput.SerializeList(outputList)`, SourceModuleId = "ContextModule"

    **System message helper:** Read from `_config.GetConfig(_context.ActiveAnimaId, "ContextModule")` with key `"systemMessage"`. Return empty string if config/context is null or key missing. Cache per call (config may change between Animas). When systemMessage is empty string, do NOT prepend system message.

    **ShutdownAsync:** Dispose all subscriptions, clear list.

    **Metadata:** Internal class `ContextModuleMetadata : IModuleMetadata` with Name="ContextModule", Version="1.0.0", Description="Manages multi-turn conversation history for LLM context".

    Thread safety note: IEventBus handlers may fire concurrently. Use `lock(_history)` around history mutations and reads. Keep lock scope minimal — do not hold lock during async I/O.
  </action>

  </verify>
