  <action>
    **Step 1 — Add `isActiveElementEditable` to editor.js:**

    Add this function to the `window.editorCanvas` object (after `getBounds`):

    ```javascript
    isActiveElementEditable: function () {
      var el = document.activeElement;
      if (!el) return false;
      var tag = el.tagName;
      if (tag === "INPUT" || tag === "TEXTAREA" || tag === "SELECT") return true;
      if (el.isContentEditable) return true;
      return false;
    },
    ```

    This checks whether the currently focused element is a text input, textarea, select dropdown, or contenteditable element. Returns `true` if the Delete key should NOT fire deletion (because user is typing in a form field).

    **Step 2 — Update `HandleKeyDown` in Editor.razor:**

    Change the `HandleKeyDown` method from synchronous `void` to `async Task`:

    ```csharp
    private async Task HandleKeyDown(KeyboardEventArgs e)
    {
        if (e.Key == "Delete" || e.Key == "Backspace")
        {
            var isEditing = await JS.InvokeAsync<bool>("editorCanvas.isActiveElementEditable");
            if (!isEditing)
            {
                _state.DeleteSelected();
            }
        }
        else if (e.Key == "Escape")
        {
            _state.ClearSelection();
        }
    }
    ```
