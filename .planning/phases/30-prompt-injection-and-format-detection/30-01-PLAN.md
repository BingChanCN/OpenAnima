<feature>
  <name>FormatDetector: XML Routing Marker Parser</name>
  <files>src/OpenAnima.Core/Modules/FormatDetector.cs, tests/OpenAnima.Tests/Unit/FormatDetectorTests.cs</files>
  <behavior>
    - Detect("plain text with no markers", {"svc"}) -> PassthroughText="plain text with no markers", Routes=[], MalformedMarkerError=null
    - Detect("Hello <route service=\"summarize\">payload</route> world", {"summarize"}) -> PassthroughText="Hello  world" (trimmed), Routes=[("summarize","payload")], MalformedMarkerError=null
    - Detect("<route service=\"Summarize\">p</route>", {"summarize"}) -> Routes[0].ServiceName="summarize" (case-insensitive match)
    - Detect("text <route service=\"svc\">p1</route> mid <route service=\"svc2\">p2</route> end", {"svc","svc2"}) -> Routes.Count=2, ordered
    - Detect("text <route service=\"svc\">multi\nline\npayload</route>", {"svc"}) -> Routes[0].Payload="multi\nline\npayload"
    - Detect("text <route service=\"unknown\">p</route>", {"svc"}) -> MalformedMarkerError contains "unknown" (unrecognized service)
    - Detect("text <route service=\"svc\">unclosed tag", {"svc"}) -> MalformedMarkerError contains "Unclosed" (unclosed tag)
    - Detect("text with <route but no closing", {"svc"}) -> MalformedMarkerError contains "Unclosed"
    - Detect("", {"svc"}) -> PassthroughText="", Routes=[], MalformedMarkerError=null
    - Detect("<route  service = \"svc\" >payload</route>", {"svc"}) -> Routes[0].ServiceName="svc" (lenient whitespace)
    - Multiple blank lines around stripped markers are normalized in PassthroughText
    - When unrecognized service found, the marker text is left in PassthroughText (not stripped)
  </behavior>
  <implementation>
    Create `FormatDetector` class in `OpenAnima.Core.Modules` namespace with:
    1. Two compiled static Regex fields:
       - `RouteMarkerRegex`: `<route\s+service\s*=\s*"([^"]*)"\s*>(.*?)</route>` with IgnoreCase | Singleline | Compiled
       - `UnclosedMarkerRegex`: `<route\b[^>]*>(?!.*</route>)` with IgnoreCase | Singleline | Compiled
    2. `Detect(string response, IReadOnlySet<string> knownServiceNames)` method:
       - First check for unclosed markers via UnclosedMarkerRegex — return error immediately
       - Then use RouteMarkerRegex.Replace to extract matches, building Routes list
       - For each match: case-insensitive lookup in knownServiceNames; if not found, set error and leave marker in text
       - Trim final passthrough text, collapse multiple consecutive blank lines to double-newline
       - Return FormatDetectionResult
    3. Records: `RouteExtraction(string ServiceName, string Payload)` and `FormatDetectionResult(string PassthroughText, IReadOnlyList<RouteExtraction> Routes, string? MalformedMarkerError)`

    File-scoped namespace: `namespace OpenAnima.Core.Modules;`
    No constructor dependencies (no ILogger needed — this is pure logic).
  </implementation>
</feature>
