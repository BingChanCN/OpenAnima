Config key conventions (from Plan 01):
- `llmProviderSlug`: provider slug string, `""` = unselected, `"__manual__"` = manual mode
- `llmModelId`: model ID under selected provider, `""` = unselected
- `apiUrl`, `apiKey`, `modelName`: existing manual fields (Group "manual")

Provider display format: `{DisplayName} ({N} models)` — e.g., "OpenAI (3 models)"
Model display format: `{modelId}` or `{modelId} ({displayAlias})` — e.g., "gpt-4o (智能模型)"
