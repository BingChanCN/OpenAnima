    1. Create `src/OpenAnima.Core/Modules/HttpRequestModule.cs` in `namespace OpenAnima.Core.Modules`:

       Port declarations:
       ```
       [InputPort("body", PortType.Text)]
       [InputPort("trigger", PortType.Trigger)]
       [OutputPort("body", PortType.Text)]
       [OutputPort("statusCode", PortType.Text)]
       [OutputPort("error", PortType.Text)]
       ```

       Constructor injection — follow AnimaRouteModule pattern exactly:
       - `IEventBus eventBus`
       - `IHttpClientFactory httpClientFactory` (NEW — not in AnimaRouteModule)
       - `IAnimaModuleConfigService configService`
       - `IAnimaContext animaContext`
       - `ILogger<HttpRequestModule> logger`
       - All with null checks via `?? throw new ArgumentNullException(...)`

       Fields:
       - `_lastBodyPayload` (string?) — buffered body from input port
       - `_subscriptions` (List of IDisposable)
       - `_state` (ModuleExecutionState)
       - `_lastError` (Exception?)

       Metadata: `new ModuleMetadataRecord("HttpRequestModule", "1.0.0", "Makes configurable HTTP requests with SSRF protection")`

       InitializeAsync:
       - Read config, if empty set defaults: `url=""`, `method="GET"`, `headers=""`, `body=""`
       - Subscribe to `{Metadata.Name}.port.body` — buffer `_lastBodyPayload`
       - Subscribe to `{Metadata.Name}.port.trigger` — call `HandleTriggerAsync(ct)`

       HandleTriggerAsync(CancellationToken ct):
       a. Set state to Running
       b. Read config: url, method, headers, body keys
       c. If url is empty/whitespace, publish error `{"error":"MissingUrl","url":""}` and return
       d. Call `SsrfGuard.IsBlocked(url, out var reason)` — if blocked, publish error `{"error":"SsrfBlocked","url":url,"reason":reason}` and return
       e. Parse headers using `ParseHeaders(raw)` helper — split on first colon per line, skip empty/invalid
       f. Determine request body: use `_lastBodyPayload` if available, otherwise fall back to config `body` value. For GET requests with no body data, body is null (acceptable).
       g. Create CancellationTokenSource AFTER SSRF check (per pitfall 4): `new CancellationTokenSource(TimeSpan.FromSeconds(10))` + `CreateLinkedTokenSource(ct, timeoutCts.Token)`
       h. Call `_httpClientFactory.CreateClient("HttpRequest")` — NEVER cache this
       i. Build `HttpRequestMessage` with configured method, URL, headers, and body (if any, as StringContent)
       j. `await client.SendAsync(request, linkedCts.Token)`
       k. On success (any HTTP status): read response body as string, publish to `{Metadata.Name}.port.body` and `{Metadata.Name}.port.statusCode` (as string, e.g. "200"). Set state Completed.
       l. Catch `OperationCanceledException when (timeoutCts.IsCancellationRequested)` — publish error `{"error":"Timeout","url":url,"timeout":10}`. Set state Error.
       m. Catch `OperationCanceledException` (heartbeat cancelled) — swallow, set state Idle.
       n. Catch `HttpRequestException ex` — publish error `{"error":"ConnectionFailed","url":url,"message":ex.Message}`. Set state Error.
       o. Catch `Exception ex` — publish error `{"error":"RequestFailed","url":url,"message":ex.Message}`. Set state Error.

       ParseHeaders helper: `private static IEnumerable<(string Name, string Value)> ParseHeaders(string raw)` — split on newlines, find first colon index, skip lines without colon, trim name and value. Per research pitfall 5: use `IndexOf(':')` not `Split(':')`.

       PublishErrorAsync: same pattern as AnimaRouteModule — serialize anonymous object to JSON, publish to error port.

       ExecuteAsync: No-op (event-driven via trigger).

       ShutdownAsync: Dispose subscriptions, clear `_lastBodyPayload`.

       GetState/GetLastError: Standard accessors.

       - Add `services.AddSingleton<HttpRequestModule>();` after the `AnimaRouteModule` line

    4. Register port metadata and auto-init — edit `WiringInitializationService.cs`:
