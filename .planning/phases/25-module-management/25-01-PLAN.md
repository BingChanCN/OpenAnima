Create IAnimaModuleStateService interface with methods:
- bool IsModuleEnabled(string animaId, string moduleName)
- Task SetModuleEnabled(string animaId, string moduleName, bool enabled)
- IReadOnlySet&lt;string&gt; GetEnabledModules(string animaId)
- Task InitializeAsync() — loads all existing enabled-modules.json files

Create AnimaModuleStateService implementation:
- Dictionary&lt;string, HashSet&lt;string&gt;&gt; _enabledModules for in-memory cache
- SemaphoreSlim(1,1) for thread-safe mutations (pattern from AnimaRuntimeManager)
- Persist to data/animas/{animaId}/enabled-modules.json using System.Text.Json
- InitializeAsync scans data/animas/ directories and loads all enabled-modules.json files
- SetModuleEnabled writes JSON after every change (immediate persistence)
- New Animas start with empty enabled set (no modules enabled by default per CONTEXT.md)

Follow Phase 23-01 patterns:
- Use System.Text.Json with JsonSerializerOptions { WriteIndented = true }
- Async file I/O (File.ReadAllTextAsync, File.WriteAllTextAsync)
- SemaphoreSlim wraps all dictionary mutations
- IAsyncDisposable for SemaphoreSlim cleanup

Write TDD tests first (RED), then implementation (GREEN).
