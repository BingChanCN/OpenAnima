   - Store `_configDirectory` and `_logger` as fields
   - Add field: `private CancellationTokenSource? _viewportDebounce`
3. Implement method `LoadAsync(string animaId, CancellationToken ct = default)`:
   - Build file path: `Path.Combine(_configDirectory, $"{animaId}.viewport.json")`
   - If file doesn't exist: return `new ViewportState()` (default: scale 1.0, panX 0, panY 0)
   - If exists: deserialize using JsonSerializer, return result or default on error
   - Log warnings on deserialization failure
4. Implement method `TriggerSaveViewport(string animaId, double scale, double panX, double panY)`:
   - Signature is async void (matches EditorStateService pattern)
   - Cancel previous debounce: `_viewportDebounce?.Cancel()` + `?.Dispose()`
   - Create new CancellationTokenSource: `_viewportDebounce = new CancellationTokenSource()`
   - Delay 1000ms: `await Task.Delay(1000, _viewportDebounce.Token)`
   - On success: create ViewportState, serialize to file, log debug
   - On OperationCanceledException: ignore (debounce was cancelled)
   - On other exceptions: log error, don't throw
5. Ensure: Directory created if missing: `Directory.CreateDirectory(_configDirectory)` if not exists
</action>
