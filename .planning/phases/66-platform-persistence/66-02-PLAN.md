<action>
1. In ChatPanel component, find or create Anima change handler:
   - If `OnAnimaChanged()` method exists, modify it
   - If not, subscribe to `_animaContext.ActiveAnimaChanged` event in OnInitializedAsync
2. In the handler, add restore logic:
   ```csharp
   private async Task RestoreChatHistoryAsync()
   {
       var activeAnimaId = _animaContext.ActiveAnimaId;
       if (string.IsNullOrEmpty(activeAnimaId))
           return;

       // Load history from database
       var messages = await _chatHistoryService.LoadHistoryAsync(activeAnimaId, CancellationToken.None);

       // Clear and repopulate UI state
       _chatSessionState.Messages.Clear();
       _chatSessionState.Messages.AddRange(messages);

       // Auto-scroll to bottom (if JS interop exists)
       await JS.InvokeVoidAsync("scrollChatToBottom", "chat-messages");
   }
   ```
3. Call this method when Anima changes: `await RestoreChatHistoryAsync()`
4. Verify: History is restored every time user switches Animas
</action>
