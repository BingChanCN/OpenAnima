**IRunRepository.cs** -- Interface:
```csharp
public interface IRunRepository
{
    Task CreateRunAsync(RunDescriptor descriptor, CancellationToken ct = default);
    Task AppendStateEventAsync(string runId, RunState state, string? reason = null, CancellationToken ct = default);
    Task<RunDescriptor?> GetRunByIdAsync(string runId, CancellationToken ct = default);
    Task<IReadOnlyList<RunDescriptor>> GetAllRunsAsync(CancellationToken ct = default);
    Task<IReadOnlyList<RunDescriptor>> GetRunsInStateAsync(RunState state, CancellationToken ct = default);
    Task<IReadOnlyList<RunStateEvent>> GetStateEventsByRunIdAsync(string runId, CancellationToken ct = default);
    Task AppendStepEventAsync(StepRecord step, CancellationToken ct = default);
    Task<IReadOnlyList<StepRecord>> GetStepsByRunIdAsync(string runId, CancellationToken ct = default);
    Task<int> GetStepCountByRunIdAsync(string runId, CancellationToken ct = default);
}
```

**RunRepository.cs** -- Implementation using Dapper:

- Constructor takes `RunDbConnectionFactory`
- Each method opens a new `SqliteConnection` via `_factory.CreateConnection()`, opens it, executes SQL, and disposes
- `CreateRunAsync`: INSERT INTO runs + INSERT INTO run_state_events (Created state)
- `AppendStateEventAsync`: INSERT INTO run_state_events; also UPDATE runs SET updated_at = @OccurredAt WHERE run_id = @RunId
- `GetRunByIdAsync`: SELECT from runs joined with MAX(id) from run_state_events to get current state. Map `state` column to `RunState` enum via `Enum.Parse<RunState>(row.CurrentStateStr)` pattern. If row not found, return null.
- `GetAllRunsAsync`: Same join pattern, ORDER BY created_at DESC
- `GetRunsInStateAsync`: Filter by current state = given state (subquery: WHERE state = @State in the MAX(id) subquery)
- `AppendStepEventAsync`: INSERT INTO step_events
- `GetStepsByRunIdAsync`: SELECT * FROM step_events WHERE run_id = @RunId ORDER BY occurred_at ASC
- `GetStepCountByRunIdAsync`: SELECT COUNT(*) FROM step_events WHERE run_id = @RunId
- `GetStateEventsByRunIdAsync`: SELECT * FROM run_state_events WHERE run_id = @RunId ORDER BY id ASC

NOTE: Dapper maps column names to property names by convention. For the current_state derived column in the join query, use a DTO class `RunRow` with all `runs` columns plus a `current_state` string column, then convert to `RunDescriptor` in the mapping step.
