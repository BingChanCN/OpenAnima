### Integration Findings

**INT-01 (Low Priority):** LLMModule's `ICrossAnimaRouter? router = null` optional parameter is resolved implicitly by .NET DI. Because `ICrossAnimaRouter` IS registered as a singleton in `AnimaServiceExtensions`, .NET's built-in DI container will resolve it correctly. However, this is fragile — recommend changing to explicit factory lambda for robustness.
