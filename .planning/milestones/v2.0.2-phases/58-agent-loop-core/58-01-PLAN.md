    **Record types** (define at top of file, outside the class):
    ```csharp
    public record ToolCallExtraction(string ToolName, IReadOnlyDictionary<string, string> Parameters);
    public record ToolCallParseResult(
        IReadOnlyList<ToolCallExtraction> ToolCalls,
        string PassthroughText,
        bool HasUnclosedMarker);
    ```

    **Static class with three compiled Regex fields:**
    1. `ToolCallRegex` — matches `<tool_call name="...">[body]</tool_call>`:
       Pattern: `@"<tool_call\s+name\s*=\s*""([^""]*)""\s*>(.*?)</tool_call>"`
       Options: `RegexOptions.IgnoreCase | RegexOptions.Singleline | RegexOptions.Compiled`

    2. `ParamRegex` — matches `<param name="...">value</param>` inside body:
       Pattern: `@"<param\s+name\s*=\s*""([^""]*)""\s*>(.*?)</param>"`
       Options: same as above

    3. `UnclosedRegex` — detects unclosed `<tool_call` tags (fast-path malformed output):
       Pattern: `@"<tool_call(?:\b[^>]*>(?![\s\S]*</tool_call>)|(?![^>]*>))"`
       Options: same as above

    **Parse method logic:**
    1. If null/empty input, return `new ToolCallParseResult([], response ?? "", false)`
    2. Check UnclosedRegex — if match, return `new ToolCallParseResult([], response, true)`
    3. Build tool call list via `ToolCallRegex.Replace(response, match => { ... })`:
       - Extract `match.Groups[1].Value` as tool name
       - Extract `match.Groups[2].Value` as body
       - Parse params from body using `ParamRegex.Matches(body)` into `Dictionary<string, string>(StringComparer.OrdinalIgnoreCase)`
       - Add `new ToolCallExtraction(name, parameters)` to list
       - Return `string.Empty` (strip marker from passthrough)
    4. Trim passthrough text
    5. Return `new ToolCallParseResult(toolCalls, passthrough, false)`

    **Class design:**
    ```csharp
    public class AgentToolDispatcher
    {
        private const int MaxToolOutputChars = 8000;

        private readonly Dictionary<string, IWorkspaceTool> _tools;
        private readonly IRunService _runService;
        private readonly ILogger<AgentToolDispatcher> _logger;
    ```

    **Constructor** accepts:
    - `IEnumerable<IWorkspaceTool> tools` — builds `_tools` dictionary keyed by `tool.Descriptor.Name` (case-insensitive: `new Dictionary<string, IWorkspaceTool>(StringComparer.OrdinalIgnoreCase)`)
    - `IRunService runService`
    - `ILogger<AgentToolDispatcher> logger`

    **DispatchAsync method:**
    ```csharp
    public async Task<string> DispatchAsync(
        string animaId,
        ToolCallExtraction toolCall,
        CancellationToken ct)
    ```
    Logic:
    1. `var runContext = _runService.GetActiveRun(animaId);`
       If null: return `FormatToolResult(toolCall.ToolName, false, "No active run — start a run first")`
    2. `if (!_tools.TryGetValue(toolCall.ToolName, out var tool))`
       Return `FormatToolResult(toolCall.ToolName, false, $"Unknown tool '{toolCall.ToolName}'. Available: {string.Join(", ", _tools.Keys)}")`
    3. Try-catch around `tool.ExecuteAsync(runContext.Descriptor.WorkspaceRoot, toolCall.Parameters, ct)`:
       - On success: `var data = ExtractDataString(result);` then `FormatToolResult(toolCall.ToolName, result.Success, data)`
       - On exception: `FormatToolResult(toolCall.ToolName, false, $"Error: {ex.Message}")`

    **CRITICAL: Does NOT touch EventBus. Does NOT acquire any semaphore.**

    **ExtractDataString method** (private):
    - If `result.Data` is `string s`, use `s`
    - Otherwise `System.Text.Json.JsonSerializer.Serialize(result.Data)` (handles anonymous objects from ToolResult.Failed)
    - If length > `MaxToolOutputChars`, truncate to `MaxToolOutputChars` and append `"\n[output truncated]"`

    **FormatToolResult method** (private static):
    ```csharp
    private static string FormatToolResult(string toolName, bool success, string content)
        => $"<tool_result name=\"{EscapeXml(toolName)}\" success=\"{success.ToString().ToLowerInvariant()}\">{EscapeXml(content)}</tool_result>";
    ```

    **EscapeXml method** (private static):
    ```csharp
    private static string EscapeXml(string value)
        => value.Replace("&", "&amp;").Replace("<", "&lt;").Replace(">", "&gt;").Replace("\"", "&quot;");
    ```
