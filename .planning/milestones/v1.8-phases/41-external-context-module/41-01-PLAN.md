    1. In PluginLoader, change `InstantiateModule` signature to pass `manifest` through to `ResolveParameter` (it already receives manifest — just thread it through the parameter resolution loop).

    2. In `ResolveParameter`, add a special case for IModuleStorage BEFORE the generic ContractsTypeMap lookup (similar to the ILogger special case at line 288). The logic:
       ```csharp
       // IModuleStorage special case - create bound instance per module
       if (paramTypeFullName == "OpenAnima.Contracts.IModuleStorage")
       {
           var unboundStorage = serviceProvider.GetService(typeof(IModuleStorage)) as ModuleStorageService;
           if (unboundStorage != null)
           {
               // Create a new bound instance using the same animasRoot/dataRoot/context
               // but with manifest.Id as boundModuleId
               var boundStorage = new ModuleStorageService(
                   unboundStorage... // need access to internal fields
               );
           }
       }
       ```

       However, ModuleStorageService fields are private. Two clean approaches:
       - (a) Add a `CreateBound(string moduleId)` method to ModuleStorageService that returns a new instance with the same animasRoot/dataRoot/context but with the given boundModuleId. This is the cleanest approach.
       - (b) Resolve IModuleContext and read animasRoot/dataRoot from config.

       Use approach (a): Add `public ModuleStorageService CreateBound(string moduleId)` to ModuleStorageService. Then in PluginLoader:
       ```csharp
       if (paramTypeFullName == "OpenAnima.Contracts.IModuleStorage")
       {
           var storage = serviceProvider.GetService(typeof(IModuleStorage)) as ModuleStorageService;
           if (storage != null && manifest != null)
           {
               return ParameterResolution.Success(storage.CreateBound(manifest.Id));
           }
           _logger?.LogWarning("IModuleStorage requested but not available or manifest missing. Passing null.");
           return ParameterResolution.Success(null);
       }
       ```

    3. Update `ResolveParameter` signature to accept `PluginManifest? manifest` parameter. Update the call site in `InstantiateModule` (line 242) to pass `manifest`.

    4. Add `CreateBound` to ModuleStorageService:
       ```csharp
       public ModuleStorageService CreateBound(string moduleId)
       {
           return new ModuleStorageService(_animasRoot, _dataRoot, _context, moduleId);
       }
       ```

    5. Add test in PluginLoaderDITests: build PortModule (which already has IModuleStorage in ContractsTypeMap), add IModuleStorage? parameter to a test module or use PortModule with an added storage param. Verify the injected IModuleStorage is bound (GetDataDirectory() does not throw).
