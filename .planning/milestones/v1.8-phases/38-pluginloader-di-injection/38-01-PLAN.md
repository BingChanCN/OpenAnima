    - LoadModule(dir, serviceProvider) with module requesting ILogger receives ILogger created via ILoggerFactory.CreateLogger(moduleType.FullName)
    - LoadModule(dir, serviceProvider) with unresolvable Contracts service parameter passes null + logs warning
    - LoadModule(dir, serviceProvider) with unresolvable optional parameter (HasDefaultValue=true) passes default value + logs warning
    - LoadModule(dir, serviceProvider) with unresolvable required parameter (no default, non-Contracts) returns LoadResult with Error
    - LoadModule(dir, null) or LoadModule(dir) with parameterless constructor module still works (backward compatibility)
    - Greedy constructor: when module has multiple constructors, the one with most parameters is selected
  </behavior>
  <action>
Replace the `Activator.CreateInstance(moduleType)` call (line 92) in PluginLoader.LoadModule with reflection-based constructor resolution. Per user decisions:

1. **Add IServiceProvider parameter** to LoadModule:
   ```csharp
   public LoadResult LoadModule(string moduleDirectory, IServiceProvider? serviceProvider = null)
   ```
   The `= null` default preserves backward compatibility for existing callers.

2. **Add ILogger field** to PluginLoader:
   ```csharp
   private readonly ILogger<PluginLoader>? _logger;
   public PluginLoader(ILogger<PluginLoader>? logger = null) { _logger = logger; }
   ```

3. **Greedy constructor selection** (replace line 92 area):
   ```csharp
   var constructors = moduleType.GetConstructors(BindingFlags.Public | BindingFlags.Instance);
   if (constructors.Length == 0)
   {
       return new LoadResult(null, context, manifest,
           new InvalidOperationException($"No public constructor found for {moduleType.FullName}"),
           false);
   }
   var selectedConstructor = constructors.OrderByDescending(c => c.GetParameters().Length).First();
   ```

4. **Parameter resolution** — add private method `ResolveConstructorParameters`:
   - For each ParameterInfo in the selected constructor:
     a. **ILogger special case**: if param.ParameterType.FullName == "Microsoft.Extensions.Logging.ILogger", resolve ILoggerFactory from serviceProvider, call CreateLogger(moduleType.FullName). If ILoggerFactory unavailable, treat as optional (null + warning per user decision).
     b. **Contracts service**: if param.ParameterType.FullName starts with "OpenAnima.Contracts.", iterate serviceProvider registered service types by getting all services and matching FullName. Use this approach: get the ServiceType from the host container by iterating `serviceProvider.GetType().GetProperties()` — actually simpler: try each known Contracts type directly:
        - Build a mapping of FullName -> host Type for the known Contracts interfaces (IModuleConfig, IModuleContext, IEventBus, ICrossAnimaRouter)
        - For each param, if its FullName matches a known Contracts FullName, call `serviceProvider.GetService(hostType)` using the HOST-side Type object
        - If resolved, use it. If null, pass null + warning (Contracts are always optional per user decision).
     c. **Unknown parameter**: if not ILogger and not Contracts:
        - If `param.HasDefaultValue` → use `param.DefaultValue` + warning log
        - Else → return LoadResult error: "Required parameter '{param.Name}' of type '{param.ParameterType.FullName}' could not be resolved"

5. **Invoke constructor**: `selectedConstructor.Invoke(resolvedArgs)`

6. **Known Contracts FullName mapping** — define as static readonly:
   ```csharp
   private static readonly Dictionary<string, Type> ContractsTypeMap = new()
   {
       ["OpenAnima.Contracts.IModuleConfig"] = typeof(IModuleConfig),
       ["OpenAnima.Contracts.IModuleContext"] = typeof(IModuleContext),
       ["OpenAnima.Contracts.IEventBus"] = typeof(IEventBus),
       ["OpenAnima.Contracts.Routing.ICrossAnimaRouter"] = typeof(ICrossAnimaRouter),
   };
   ```
   This avoids iterating all registered services — we know exactly which Contracts types to look up.

7. **Also update ScanDirectory** to accept and pass through IServiceProvider:
   ```csharp
   public IReadOnlyList<LoadResult> ScanDirectory(string modulesPath, IServiceProvider? serviceProvider = null)
   ```
   Pass serviceProvider to each LoadModule call.

Important: Do NOT use `Type.IsAssignableFrom` for cross-context matching — always use FullName string comparison. Do NOT use generic `GetService<T>()` — use non-generic `GetService(Type)` with host-side Type objects from ContractsTypeMap.
