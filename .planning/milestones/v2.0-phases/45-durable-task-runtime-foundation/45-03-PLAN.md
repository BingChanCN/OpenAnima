**`@code` block structure:**

Fields:
- `private List<RunDescriptor> _runs = new();`
- `private Dictionary<string, string?> _stopReasons = new();` -- runId -> reason
- `private Dictionary<string, int> _stepCounts = new();` -- runId -> count
- `private HubConnection? _hubConnection;`
- `private bool _showCancelConfirm;`
- `private string? _cancelTargetRunId;`
- `private string? _startError;`

`OnInitializedAsync()`:
1. Load runs: `_runs = (await RunService.GetAllRunsAsync()).ToList();`
2. For each run, load step counts from the service (if available) -- or initialize to 0
3. Set up SignalR HubConnection:
```csharp
_hubConnection = new HubConnectionBuilder()
    .WithUrl(Navigation.ToAbsoluteUri("/hubs/runtime"))
    .WithAutomaticReconnect()
    .Build();

_hubConnection.On<string, string, string, string?>("ReceiveRunStateChanged",
    async (animaId, runId, state, reason) =>
    {
        // Update the run's state in the local list
        var idx = _runs.FindIndex(r => r.RunId == runId);
        if (idx >= 0)
        {
            var parsed = Enum.Parse<RunState>(state);
            _runs[idx] = _runs[idx] with { CurrentState = parsed };
            if (reason != null)
                _stopReasons[runId] = reason;
        }
        else
        {
            // New run -- reload from service
            var updated = await RunService.GetAllRunsAsync();
            _runs = updated.ToList();
        }
        await InvokeAsync(StateHasChanged);
    });

_hubConnection.On<string, string, string, string, string, int?>("ReceiveStepCompleted",
    async (animaId, runId, stepId, moduleName, status, durationMs) =>
    {
        _stepCounts[runId] = _stepCounts.GetValueOrDefault(runId) + 1;
        await InvokeAsync(StateHasChanged);
    });

await _hubConnection.StartAsync();
```

`HandleStartRun` method (callback for RunLaunchPanel):
1. Get active anima ID from `AnimaContext.ActiveAnimaId`
2. Call `RunService.StartRunAsync(animaId, objective, workspace, maxSteps, maxWallSeconds)`
3. If result.IsSuccess: clear error, reload runs list
4. If !result.IsSuccess: set `_startError`

`HandlePauseRun(string runId)`:
1. Call `RunService.PauseRunAsync(runId, "User paused")`
2. Reload runs list

`HandleResumeRun(string runId)`:
1. Call `RunService.ResumeRunAsync(runId)`
2. Reload runs list

`HandleCancelRequest(string runId)`:
1. Set `_cancelTargetRunId = runId`, `_showCancelConfirm = true`

`HandleCancelConfirm()`:
1. Call `RunService.CancelRunAsync(_cancelTargetRunId!)`
2. `_showCancelConfirm = false`
3. Reload runs list

`HandleCancelDismiss()`:
1. `_showCancelConfirm = false`, `_cancelTargetRunId = null`

`DisposeAsync()`:
1. If `_hubConnection != null`, `await _hubConnection.DisposeAsync()`
